# NOSQL_Mongo_DB_ASSIGNMENT_Project
Overview
This project uses MongoDB to manage a database with students, faculty, courses, enrollments, and activities. It demonstrates NoSQL concepts like queries, aggregation, and CRUD operations.

Collections
students faculty courses enrollments activities

Setup
Open MongoDB Compass Connect: mongodb://localhost:27017 Create DB: Mongo_assignment Add collections & insert sample data

Features
Complex queries (filters & projections) Joins using $lookup Aggregations (average, count) CRUD operations (update, delete, upsert) Sorting & grouping

Tech Used
MongoDB MongoDB Compass

Author
Suraj Prajapati

Conclusion
Shows how MongoDB handles data efficiently using NoSQL techniques.
